# informatics-tasks

Репозиторий с заданиями по дисциплине "Анализ данных в строительстве".

## Структура

- `task_01_passport.py` — Цифровой паспорт строительного объекта
- `task_02_room_calc.py` — Параметры помещения
- `task_03_estimate_compare.py` — Конвертер температур
- `task_04_foundation_load.py` — Рабочий график
- `task_05_materials_vat.py` — Калькулятор скидки
- `task_06_materials_catalog.py` — Каталог материалов
- `task_07_price_dict.py` — Прайс-лист материалов
- `task_08_set_analysis.py` — Анализ заказов
- `task_09_address_clean.py` — Очистка адресов
- `task_10_warehouse_system.py` — Система учёта склада
